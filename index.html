Basic Features of CATaLog


  • A new web-based post-editing tool, enhanced with a number of new features.
  • Can be used as a common CAT tool but also for post-editing TM segments or MT output.
  • Provides a complete set of log information currently not available in most CAT tools.
  • Convenient for translation process research.
